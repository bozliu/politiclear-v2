# Feedback Guide

Politiclear V2 is public-facing. The most useful feedback is concrete, source-linked, and reproducible.

## Use GitHub Issues For

- broken links to official or party sources
- wrong constituency matches
- stale candidate office, committee, question, debate, or vote records
- misleading summaries or source labels
- accessibility blockers
- mobile or desktop layout regressions

## Include These Details

- page or feature
- constituency or candidate involved
- the source URL that looks wrong or failed
- what you expected to happen
- steps to reproduce
- screenshots when possible

## Reporting path

Target repo issues:

- [Public GitHub Issues](https://github.com/bozliu/politiclear-v2/issues/new/choose)

If the final repo slug changes, update:

- `src/config/projectLinks.js`
- `.github/ISSUE_TEMPLATE/config.yml`
- this file
